## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for loading Google Fonts and external services)

### Local Setup
```bash
# Clone the repository
git clone https://github.com/sunnyshah7/portfolio.git
cd portfolio

# Open in browser (no build required!)
open index.html
# or
python -m http.server 8000
# Visit http://localhost:8000
```

### Customization

#### Edit Your Information
Open `index.html` and update:
1. **Hero Section** — Name, title, summary
2. **About Section** — Background, certifications
3. **Skills Section** — Technology stack
4. **Experience Section** — Work history
5. **Projects Section** — Showcase your work
6. **Contact Section** — Email, phone, social links

#### Customize Styling
Edit CSS variables in the `<style>` section:
```css
:root {
  --bg: #080C14;              /* Main background */
  --green: #00FF88;           /* Accent color */
  --blue: #4D9FFF;            /* Secondary accent */
  --text: #E8EDF5;            /* Text color */
  --muted: #8494A8;           /* Muted text */
  /* ... more variables */
}
```

#### Change Colors
Replace the hex color values to match your brand:
- Primary accent: `--green`
- Secondary accent: `--blue`
- Background: `--bg`, `--surface`
- Text: `--text`, `--muted`
